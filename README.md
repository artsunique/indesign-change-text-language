# InDesign Change Text Language

**DE** | [EN](#english)

---

## Deutsch

Ein Adobe InDesign Script zum Ändern der Sprache aller Textbereiche im Dokument.

### Features

- Dropdown mit allen installierten Sprachen/Wörterbüchern
- Häufig verwendete Sprachen (DE, EN, ES, FR, IT, etc.) oben sortiert
- Ändert Rechtschreibprüfung und Silbentrennung
- Optionen:
  - Lokale Formatierungen einbeziehen
  - Tabellen einschließen
- Detaillierter Ergebnisbericht

### Installation

1. Script herunterladen (`ChangeTextLanguage.jsx`)
2. In den InDesign Scripts-Ordner kopieren:
   - **macOS:** `~/Library/Preferences/Adobe InDesign/[Version]/[Sprache]/Scripts/Scripts Panel/`
   - **Windows:** `C:\Users\[User]\AppData\Roaming\Adobe\InDesign\[Version]\[Sprache]\Scripts\Scripts Panel\`
3. In InDesign: Fenster → Hilfsprogramme → Skripte
4. Doppelklick auf das Script

### Verwendung

1. Dokument öffnen
2. Script starten
3. Zielsprache aus Dropdown wählen
4. Optional: Optionen anpassen
5. "Ändern" klicken

### Anwendungsfall

Ideal für mehrsprachige Projekte, bei denen ein Dokument in verschiedene Sprachen übersetzt wird und die Spracheinstellung für Rechtschreibprüfung/Silbentrennung angepasst werden muss.

## English

An Adobe InDesign script to change the language of all text areas in a document.

### Features

- Dropdown with all installed languages/dictionaries
- Common languages (DE, EN, ES, FR, IT, etc.) sorted at top
- Changes spell checking and hyphenation settings
- Options:
  - Include local overrides
  - Include tables
- Detailed result report

### Installation

1. Download the script (`ChangeTextLanguage.jsx`)
2. Copy to InDesign Scripts folder:
   - **macOS:** `~/Library/Preferences/Adobe InDesign/[Version]/[Language]/Scripts/Scripts Panel/`
   - **Windows:** `C:\Users\[User]\AppData\Roaming\Adobe\InDesign\[Version]\[Language]\Scripts\Scripts Panel\`
3. In InDesign: Window → Utilities → Scripts
4. Double-click the script

### Usage

1. Open a document
2. Run the script
3. Select target language from dropdown
4. Optional: Adjust options
5. Click "Ändern" (Change)

### Use Case

Ideal for multilingual projects where a document is translated into different languages and the language setting for spell checking/hyphenation needs to be adjusted.
